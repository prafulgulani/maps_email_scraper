# Maps and Email Extractor

A simple lead-generation toolkit that:

1. Scrapes businesses from Google Maps  
2. Visits each website and extracts emails + social media links  
4. Sorts the data on rating and ratings count and cleans, sorts, and removes duplicate contacts  

Built with **Python + Playwright + Pandas**.

---

## Getting Started

# Clone the repository
git clone https://github.com/prafulgulani/maps_email_scraper.git
cd maps_email_scraper

# Create and activate virtual environment
python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate

# Install dependencies
pip install -r requirements.txt

# Install Playwright browsers
playwright install

### Execution Steps

After setting up the environment, follow this pipeline to gather and clean your data:
1. Configure Search Area and run the maps_scraper
Open maps_scraper.py and update the latitude and longitude coordinates in the generate grid funcion to define your search area and then run the file.

2. Run the email scraper
Pass the csv file generated by maps scraper to the email_scraper.py and execute the script

3. Data Post-Processing
Pass the csv generated by email_scraper to clean_sort.py to get the final csv without dupplicates and in sorted order
